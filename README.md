# Instaphoto-Login
*Module for a log-in user interface.*

## List of implementations
* MVVM and Clean architecture.
* Jetpack Compose.
* Retrofit for API call.
* Gson for response parsing.
* Dependency injection with Dagger Hilt.
* LiveData.
